Local font bundle for offline and restricted-network use.

Downloaded from official repositories:

- Google Fonts repo: https://github.com/google/fonts
- Noto CJK repo: https://github.com/notofonts/noto-cjk

Included families:

- BIZ UDGothic
- Cormorant Garamond
- Hina Mincho
- Kosugi Maru
- LINE Seed JP
- Noto Sans JP
- Noto Serif JP
- Sawarabi Mincho
- Shippori Mincho
- Zen Kaku Gothic New
- Zen Old Mincho
